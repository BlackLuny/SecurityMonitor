int test_gpaToHPA ( )

A testing function for gpaToHPA.

Details.

This function tests gpaToHPA function. The specification of this function is as follows:

  1. Input value : GPA of currently running process's page table base.
  2. Expected output : In bitvisor, the GPA is mapped the same as the HPA. Therefore, the output should be the same with the input value.

void test_start ( )

A function to start unit test.

Details.

This function starts unit testing. To be specific, this function invokes test_* functions.
These test_* functions returns boolean (actually, integer) value. Based on the test result, this function prints the testing output.

int test_traverseGuestPages ( )

A testing function for traverseGuestPages.

Details.

This function tests traverseGuestPages function. The purpose of this function is to check whether this function traverses all pages of the currently running process.
This function has no automatic testing method. So developers have to verify the printed results. For it this function prints GPA of each page by traversing the pages using printPageGPA function.
