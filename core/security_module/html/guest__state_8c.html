Traverse all pages that start from startGPAofPageTable and do something for each page.

Detail.{traverseGuestPages} This function does two things: 1. This function finds all pages of a process whose page table starts from the input *startGPAofPageTable*. 2. For each page, if needed, call a function named *do_something* passing VMID, APPID, and GPA of a page as a parameter. What the function does is depends on the caller.

주의:
In this implementation, the security monitor is assuming that the guest is using IA32e paging mode.
매개변수:
[in]vmidVM ID to be passed to the do_something()
[in]appIDAPP ID to be passed to the do_something()
[in]startGPAofPageTablestart GPA of a process's page table
[in]do_somethinga function pointer that will be invoked for each page
