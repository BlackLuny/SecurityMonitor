int test_gpaToHPA ( )

A testing function for [gpaToHPA](gpaToHPA)

Details.

This function tests [gpaToHPA](gpaToHPA) function. The specification of this function is as follows: 1. Input value : GPA of currently running process's page table base. 2. Expected output : In bitvisor, the GPA is mapped the same as the HPA. Therefore, the output should be the same with the input value.

unit_test.c 파일의 37 번째 라인에서 정의되었습니다.

{
    GPA_t input;
    HPA_t output;
    HPA_t expected;
    input = (GPA_t)get_page_table_base_GPA();
    input |= 0xCDE;
    output = gpaToHPA(input, 0);
#ifdef CONFIG_BITVISOR
    expected = input;
    return expected == output;
#else
    expected = 0;
    return 0;
#endif
}

void test_start ( )

A function to start unit test.

Details.

This function starts unit testing. To be specific, this function invokes test_* functions.
These test_* functions returns boolean (actually, integer) value. Based on the test result, this function prints the testing output.

int test_traverseGuestPages ( )

A testing function for [traverseGuestPages](traverseGuestPages)

Details.

This function tests [traverseGuestPages](traverseGuestPages) function. The purpose of this function is to check whether this function traverses all pages of the currently running process.
This function has no automatic testing method. So developers have to verify the printed results. For it this function prints GPA of each page by traversing the pages using [printPageGPA](printPageGPA) function.
