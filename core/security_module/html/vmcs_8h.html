Definition of constants related to VMCS.

# Details This file defines several constants that are used to read a value from and write a value to the VMCS.
Most of these constnts are field encodings in VMCS.
Read __Intel's architecture manual Volume 3, APPENDIX B : Field Encoding in VMCS.__

U64_t get_guest_cr3_value ( )

get the value of the guest CR3 control register

반환값:
This function returns guest's CR3 control register value.

Details.

This function returns the value of CR3 control register by reading VMCS.

경고:
The return value of this function does not specifies the exact address of the page directory.
> Bits [11:0] of the return value are used to specify some information. Read __Intel's architecture manual Vol. 3, Chapter 2.5 : Control Registers__.
